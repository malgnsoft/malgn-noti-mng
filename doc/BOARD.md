# 맑은노티 현황판 — Step 5 서비스 개발 (정본)

> **목적**: 현황판([`/board`](https://malgn-noti-mng.pages.dev/board))의 **Step 5 · 서비스 개발** 세부 항목을 큰 카테고리로 묶어 **단순화·재정의**한 정본 문서.

> **마지막 현행화**: 2026-06-05

## 범례 · 재정의 원칙

- 상태: ✅ 완료 · 🟡 진행 중 · ⚪ 대기
- 기존 잘게 쪼갠 92개 세부 task를 **영역별 큰 카테고리**로 묶었다. 카테고리 상태는 하위 항목 종합.
- 괄호 안 `5-2-1` 등은 기존 WBS task id (추적용). 간트 WBS로 옮길 때 매핑 근거.
- 단순화 대상 4개 영역: **API 서버**, **API 엔드포인트**, **사용자단 ↔ API 연동**, **관리자단 화면**.
  - 기존 `API 서버`(16) + `API 엔드포인트`(5)는 성격이 같아 **하나의 "API 백엔드"로 통합**.
