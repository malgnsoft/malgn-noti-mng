## 1. 개요

발송 후 **결과를 조회·분석하는 도메인**. 5개 채널별 이력 조회 + 통합 통계 대시보드 제공.

**핵심 기능**:
- 채널별 발송 이력 테이블 (필터/정렬/다운로드)
- 예약 발송 일괄 취소
- 통계 대시보드 (채널별/일별 추이 · 실패 분석)

### 4.2 테이블 컴포넌트 (AppHistoryView)

**고정 컬럼** (좌→우):

| # | 컬럼 | 타입 | 설명 |
| --- | --- | --- | --- |
| 1 | ID | 버튼 | 발송 ID (클릭 시 상세 팝업) |
| 2 | 요청 일시 | 텍스트 | `YYYY-MM-DD HH:MM:SS` |
| 3 | 발송 시점 | 배지 | `즉시` / `예약` |
| 4 | 발신자 | 텍스트 | 채널별 발신 정보 (번호/도메인/프로필) |
| 5 | 발송 상태 | 배지 | `발송 성공` · `발송 실패` · `발송 중` · `발송 취소` · `발송 예약` 등 |
| 6 | 발송 목적 | 텍스트 | `일반용` / `인증용` / `광고용` |
| 7 | 수신자 | 텍스트 | 수신자 정보 (폰번호/이메일/토큰 등, 마스킹) |
| 8 | 수신 상태 | 배지 | `수신 완료` / `수신 대기` / `실패` |

**선택 추가 컬럼** (체크박스로 on/off):

```
[✓] 예약 일시  [✓] 발송 일시  [✓] 수신 일시  
[✓] 템플릿  [✓] 플로우
```

- 예약 일시: `YYYY-MM-DD HH:MM:SS`
- 발송 일시: 실제 발송된 시각
- 수신 일시: 수신 완료된 시각 (또는 "-")
- 템플릿: 적용된 템플릿 이름 (또는 "-")
- 플로우: 해당하는 Flow 이름 (또는 "-")

### 4.4 행 액션

#### 행 클릭 → 상세 팝업

| 항목 | 내용 |
| --- | --- |
| **크기** | 800px(width) · 80vh(height) |
| **콘텐츠** | 발송 ID · 요청/발송/수신 일시 · 발신자 · 수신자(전체 공개) · 메시지 미리보기(채널별) · 발송 상태 상세(성공/실패 코드) · 재발송 버튼(실패 시) |

#### 일괄 액션 (체크박스 선택 시)

좌측 상단에 Bulk Action Bar 노출:

```
[선택: 5개] [취소 (예약건만)] [CSV] [삭제(미전송)]
```

- **취소**: 발송 상태가 `예약` 또는 `대기`인 건만 취소 가능 → 확인 팝업(`AppRiskActionDialog`) → API `/v1/send/{id}/cancel`
- **CSV**: 선택 행만 다운로드
- **삭제**: 발송 상태가 `요청` 또는 `취소`인 건만 삭제 가능 → 위험 팝업

## 7. 데이터 모델

### 발송 이력 행 (Row)

```typescript
interface HistoryRow {
  id: string                          // 발송 ID (UUID)
  channelType: 'sms' | 'rcs' | 'kakao' | 'email' | 'push'
  requestedAt: ISO8601                // 요청 일시
  timing: 'now' | 'schedule'          // 즉시/예약
  sender: string                      // 발신 정보 (번호/도메인/프로필)
  sendStatus: 'requested' | 'pending' | 'in_progress' | 'success' | 'failed' | 'cancelled'
  purpose: 'info' | 'auth' | 'ad'    // 발송 목적
  recipient: string                   // 수신자 (마스킹)
  receivedStatus: 'pending' | 'success' | 'failed'
  reservedAt?: ISO8601               // 예약 발송 시각
  sentAt?: ISO8601                   // 실제 발송 시각
  receivedAt?: ISO8601               // 수신 완료 시각
  templateName?: string              // 템플릿 이름
  flowName?: string                  // 플로우 이름
}
```

### 통계 데이터 (KPI)

```typescript
interface StatsSummary {
  range: '1d' | '7d' | '30d'
  totalSend: number
  totalSuccess: number
  totalFailed: number
  failureRate: number                 // %
  byChannel: {
    sms: { send: number, success: number, failed: number }
    rcs: { send: number, success: number, failed: number }
    // ...
  }
}
```
