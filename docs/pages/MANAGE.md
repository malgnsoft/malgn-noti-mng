## 1. 개요

고객사 사용자가 **자신의 메시지 템플릿을 채널별로 관리하는 도메인**. 발송 시 템플릿을 선택하여 빠르고 일관된 메시지를 전송할 수 있다.

**핵심 기능**:
- 채널별 템플릿 CRUD (SMS/RCS/Kakao/Email/Push)
- 템플릿 카테고리/분류
- 샘플 템플릿 (플랫폼 제공)
- AI 템플릿 생성 (키워드 입력 → 자동 초안)
- 슬롯 변수 관리 (#{var})
- 발송 상세 설정 (대체문자·링크 추적 등)

## 14. 데이터 모델

### 템플릿 (Template)

```typescript
interface Template {
  id: string
  tenantId: string
  name: string
  channel: 'sms' | 'rcs' | 'kakao' | 'email' | 'push'
  
  categoryId?: string
  description?: string
  
  // 채널별 필드
  sms?: {
    type: 'sms' | 'lms' | 'mms'
    subject?: string              // LMS/MMS만
    body: string
    attachments?: { name: string, size: number }[]  // MMS만
  }
  rcs?: {
    imageUrl?: string
    title?: string
    body: string
    buttons?: { text: string, action: string, value: string }[]
  }
  kakao?: {
    type: 'alimtalk' | 'friendtalk'
    profileId: string
    body: string
  }
  email?: {
    subject: string
    senderName?: string
    body: string                  // HTML
    attachments?: { name: string, size: number }[]
  }
  push?: {
    title: string
    body: string
    imageUrl?: string
    landingUrl?: string
    badge?: number
    silent?: boolean
  }
  
  // 변수
  variables?: {
    name: string
    defaultValue?: string
    required: boolean
  }[]
  
  // 메타
  createdAt: ISO8601
  updatedAt: ISO8601
  usageCount?: number            // 발송 시 사용된 횟수
}
```

```typescript
  id: string

  name: string
  description?: string
  channel: 'sms' | 'rcs' | 'kakao' | 'email' | 'push'
  
  createdAt: ISO8601
}
```
